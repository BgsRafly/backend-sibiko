# SI-BIKO (Sistem Informasi Bimbingan Konseling)

SI-BIKO adalah platform manajemen bimbingan konseling berbasis API yang dirancang untuk memfasilitasi penanganan masalah mahasiswa secara berjenjang, mulai dari tingkat Program Studi (Dosen PA/Konselor) hingga tingkat Fakultas (Wakil Dekan 3).

- **Automated Handler**: Ajuan mahasiswa otomatis diarahkan ke Dosen PA masing-masing berdasarkan database.
- **Referral System**: Pencetakan data rujukan otomatis untuk penanganan tingkat Universitas.
- **Admin Dashboard**: Statistik global dan manajemen data User (Mahasiswa & Staff) secara terpusat.
